เกี่ยวกับฉัน

  สวัสดีครับ ผม buy_yakhoo กำลังศึกษาอยู่ในชั้นมัธยมศึกษาปีที่ 5 สนใจภาษา Python, C, JS เทคโนโลยีเป็นสิ่งที่ผมให้ความสนใจมากๆ ซึ่งเป็นจุดๆ หนึ่งที่ผมอยากจะทำคอนเท้นต์เกี่ยวกับสิ่งนี้ ผมคิดว่าเทคโนโลยีนั้นจะช่วยในเรื่องความสะดวก การพัฒนาการศึกษาอย่างไม่หยุดยั้ง ความคิดสร้างสรรค์ในการสร้างสิ่งใหม่ๆ

แรงบันดาลใจ

   มีแรงบันดาลใจจากการที่ผมเล่นเกม Watch Dogs ซึ่งเป็นจุดเริ่มต้นของการศึกษาการแฮก ซึ่งเป็นจุดหนึ่งที่ผมสนใจเกี่ยวกับการเขียนโปรแกรม ข่าวจาก Social media, แม้กระทั่งพี่ชายของผมเอง ที่เขียนโค้ดเก่งๆ สมัยตอนที่ผมยังเด็ก

ทำไมถึงอยากทำเว็บล่ะ?

  เพราะสมัยนี้ความรู้สามารถหาได้จากการเข้าเว็บต่างๆ ซึ่งต้องสร้างเว็บขึ้นมา และการสร้างเว็บก็ต้องเรียนรู้เกี่ยวกับการเขียน HTML, CSS, JS หรือ Framework ต่างๆ

เป้าหมาย

   เป้าหมายคือการเรียนรู้เพื่อนำไปต่อยอด เข้าคณะที่อยากเข้าอย่างเช่น Computer Science แล้วไปทำงานกับบริษัทต่างชาติ และในขณะเดียวกับก็จะทำ Project ส่วนตัวที่เป็น Open Source ซักอย่างเพื่อเป็นประโยชน์ของการพัฒนาต่อไป
